# Number-Guessing

A simple number guessing game that I made in order to complete my list of Python projects

## Usage
- Python
    - `python numberguessing.py`
- Python 3
    - `python3 numberguessing.py`

## How It Works
You have 3 chances to guess the random number, that's all!
